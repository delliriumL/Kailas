# Kailas

Сайт Kailas-lab — Next.js (App Router), Tailwind CSS.

## Локально

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Деплой на Vercel

Подключите репозиторий GitHub в [Vercel Dashboard](https://vercel.com/new) — корень проекта: этот каталог. Framework Preset: Next.js.
